Atenção para os pacotes a serem utilizados neste projeto

 E importante assim que criar o ambiente do Nodes no projeto e preciso que seja instalado os pacotes abaixo:


################################################### Extensões a serem instaladas para Utilizar no APP ###################################################

//Este comando serve para instalar a extensão que sera utilizada para fazer a conexão com o banco de dados SQL SERVER
npm install -S mssql

//Este comando serve para instalar a extensão que funcionara em conjunto com a mssql para fazer a conexão com o banco de dados
npm install msnodesqlv8

//Este pacote serve para ir atualizando em tempo real a sua aplicação assim que as alterações forem detectadas
npm install -g nodemon

//Framework usado para criar as paginas customizadas  e sera utilizado para criar a API da aplicação
npm install -S express body-parser
